hexo + GitHub个人博客网站

hexo简介

快速、简洁且高效的博客框架

前置工作

  1. 安装git

    1. 下载地址https://www.git-scm.com/download

    2. 双击下载的exe文件,一路next,直到安装完成,安装成功后,桌面点击右键,会多出两个栏目,分别是Git Bash Here/Git Gui Here

    3. 桌面点击右键,点击git bash here,进入命令行模式以下命令

    4. 执行完上一步以后,会在./ssh目录下生成公司秘钥,留作后边用,git安装就完成了

  2. 安装node.js

    1. 下载地址https://nodejs.org/zh-cn/
    3. 下载完以后,点击下载的exe文件,一路next,完成安装
    4. 安装完成后,cmd执行node -v/ npm -v 检查安装是否成功

  3. 配置GitHub

    1. 注册账号
    2. 新建仓库,仓库名:账户名.github.io,勾选Initialize this repository with a README,会默认启用github pages
    3. 点击仓库setting,下拉,会看到pages地址,可直接访问
    5. 访问出现如下界面，pages即配置完毕
    6. 个人信息中，配置ssh页面，新加ssh，将1.4生成的id_rsa.pub文件中的内容复制到key中保存
    7. 记录仓库地址

本地安装

  1. 在本地电脑适当位置新建blog的空目录
  2. cmd进入目录
  3. 执行npm install hexo-cli -g 安装hexo依赖
  4. 执行hexo init 初始化hexo项目
  5. 执行npm install 安装node依赖
  6. 执行hexo server 启动本地服务器
  7. http://localhost:4000/访问本地博客,出现如下界面说明安装成功![](E:\bolg\img\local.png)

部署至GitHub

  1. 更改blog目录下_config.yml配置文件中deploy配置

  2. node 部署插件 npm install hexo-deployer-git –save

  3. 执行部署命令 hexo d -g

  4. 部署成功后访问你的地址：http://用户名.github.io即可看到

更换主题

  1. 在博客文件夹中点击git bash here,执行git clone https://github.com/iissnan/hexo-theme-next themes/next
